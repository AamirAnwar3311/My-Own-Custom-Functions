## [1.0.0] - 2024-12-20

### Added

- Initial release of HelperFunctions package
- **ApiHelper**: Complete API request functionality with GET, POST, PUT, DELETE, PATCH support
- **MathHelper**: Mathematical operations including addition, subtraction, multiplication, division
- **StringHelper**: String manipulation utilities including slug creation, OTP generation, masking, truncation
- **DateHelper**: Date operations using Carbon including expiry checks, formatting, age calculation
- **FileHelper**: File upload and management with image validation
- **ArrayHelper**: Array manipulation utilities including dot notation access, grouping, plucking
- **ValidationHelper**: Input validation for emails, URLs, IPs, credit cards, passwords, and more
- Laravel Service Provider with singleton bindings
- Comprehensive documentation and usage examples
- MIT License

### Features

- PSR-4 autoloading
- Laravel auto-discovery support
- All methods are static for easy access
- Comprehensive PHPDoc comments
- Error handling and validation
- Type hints and return types
- Backward compatible with original helper functions
